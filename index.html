Virtual Reality teaching kit

using WebVR and A-frame

Introduction

Virtual Reality is the current most exciting part of technology. It is a way to create immersive experiences for a variety of applications ranging from gaming to education to simulation.

However, building VR experiences is often considered as a complex task, mainly due to the tremendous assets and features it seems to carry.

With this activity you will teach the easiest way to create VR experiences, using AFrame. You will also explain how to play Virtual Reality on the web, of course, with all the benefits of it, using WebVR. Now, off to buiding the first VR scene !.

Mini Hackathon

If you are following the default presentation guide, you will most likely conduct a min-hackathon at the end of your presentation. It's always the best way to get the audience excited and let them to code hands-on under the supervision of a facilitator (that's you !)

Break the audience into small groups:

  • The goal is to build a Virtual Reality scene.
  • The audience can use the a-frame documentation and the rest of the web
  • The team that creates the best scene, wins. The evaluation can be done based on the use of various components and primitives from the documentation. Extra marks if the team uses models and other advanced objects.

Make sure to share the best makes with the mozilla campus team, post the event!

Why do this activity?

Mozilla cares for a better and open web. WebVR is an open source technology using which the advantages of the web can be explored for building VR. The work is further simplified with A-Frame. I think every developer should know the existence of such an amazing technology !
